# Advanced Sudoku 🎮

A next-generation Sudoku experience built with Flutter.

## Features
- Multiple Difficulty Levels
- Intelligent Sudoku Generator
- Hint System
- Timer & Statistics
- Daily Challenges
- Achievements & Rewards
- Global Leaderboards
- Player Analytics Dashboard
- Modern Neon Gaming UI

## Tech Stack
- Flutter
- Dart
- Material 3
